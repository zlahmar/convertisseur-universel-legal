📱 Description du service

UniConvert est une application mobile gratuite permettant de convertir des fichiers entre différents formats :

  • Image → PDF
  • Texte → PDF
  • Scan → PDF
  • Image → Texte (OCR)
  • PDF → Image
  • Vidéo → Audio (bientôt disponible)
  • Audio → Texte (bientôt disponible)
