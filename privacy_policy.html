🔒 Politique de confidentialité

Convertisseur Universel — Dernière mise à jour : avril 2026

📌 Introduction

La présente politique de confidentialité décrit comment l'application Convertisseur Universel, développée par l'équipe Convertisseur Universel, collecte, utilise et protège vos données lorsque vous utilisez notre application mobile.

En utilisant cette application, vous acceptez les pratiques décrites dans cette politique.

📁 Données collectées

Notre application est conçue pour respecter votre vie privée. Voici ce que nous collectons ou non :

  • Fichiers : Les fichiers que vous convertissez (images, PDF, textes, vidéos) sont traités uniquement en local sur votre appareil. Ils ne sont jamais envoyés sur nos serveurs.
  • Données personnelles : Nous ne collectons aucune donnée personnelle identifiable (nom, email, localisation).
  • Données publicitaires : Notre partenaire publicitaire Google AdMob peut collecter des données anonymes pour afficher des publicités personnalisées (voir section AdMob).

📱 Traitement local des fichiers

Toutes les conversions effectuées dans l'application sont réalisées entièrement sur votre appareil :

  • Image → PDF : traitement local
  • Texte → PDF : traitement local
  • Scan → PDF : traitement local
  • Image → Texte (OCR) : traitement local via Google ML Kit
  • PDF → Image : traitement local
  • Vidéo → Audio : cette fonctionnalité utilisera une API tierce (Convertio). Dans ce cas, le fichier vidéo sera temporairement envoyé sur les serveurs de Convertio pour traitement, puis supprimé. Vous en serez informé dans l'application.

📢 Publicités — Google AdMob

Notre application utilise Google AdMob pour afficher des publicités. AdMob peut collecter les données suivantes :

  • Identifiant publicitaire de l'appareil
  • Données d'utilisation anonymes
  • Informations sur l'appareil (modèle, système d'exploitation)

Ces données sont utilisées pour afficher des publicités pertinentes. Pour en savoir plus sur la manière dont Google utilise ces données, consultez : policies.google.com/privacy

Vous pouvez désactiver la personnalisation des publicités dans les paramètres de votre appareil Android : Paramètres → Google → Annonces → Désactiver la personnalisation des annonces

🔐 Permissions utilisées

L'application demande les permissions suivantes :

  • Caméra : pour scanner des documents et prendre des photos à convertir
  • Stockage (lecture/écriture) : pour accéder à vos fichiers et sauvegarder les fichiers convertis

Ces permissions sont utilisées uniquement pour les fonctionnalités de conversion. Aucune donnée n'est partagée avec des tiers sans votre consentement.

👶 Protection des mineurs

Notre application ne collecte pas sciemment de données concernant des enfants de moins de 13 ans. Si vous pensez qu'un enfant nous a fourni des données personnelles, contactez-nous pour que nous les supprimions.

🔄 Modifications de cette politique

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page avec une date de mise à jour. Nous vous encourageons à consulter cette page régulièrement.

📬 Contact

Pour toute question concernant cette politique de confidentialité, vous pouvez nous contacter via la page du Play Store ou en ouvrant une issue sur notre dépôt GitHub.
